# 🎬 视频网站 - 使用指南

## ✅ 服务器状态：正常运行！

### 🚀 快速开始
1. **启动服务器**：双击 `D:\666\start-universal.bat`
2. **等待启动**：看到"服务器已启动"提示
3. **打开浏览器**：访问 `http://localhost:8888`

### 📱 访问地址
- **通用访问**：http://localhost:8888
- **电脑版**：http://localhost:8888/demo.html
- **手机版**：http://localhost:8888/demo-mobile.html
- **测试页面**：http://localhost:8888/test.html

### 🎯 功能特性
- 🔐 **用户系统**：注册、登录、个人中心
- 🎬 **视频浏览**：热门视频、分类浏览
- 🔍 **搜索功能**：关键词搜索、分类筛选
- 📤 **上传功能**：视频上传（演示版）
- 📊 **个人统计**：观看记录、收藏管理

### 📱 设备适配
- **自动检测**：系统自动识别手机/电脑
- **响应式设计**：完美适配各种屏幕尺寸
- **触控优化**：手机版支持触控操作

### 🔧 故障排除
如果无法访问：
1. 检查Node.js是否安装
2. 确保端口8888未被占用
3. 检查防火墙设置
4. 重新启动服务器

### 🎉 现在就可以使用了！
打开浏览器访问 http://localhost:8888 开始体验视频网站！
