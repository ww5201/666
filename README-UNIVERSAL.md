# 🎬 视频网站使用说明

## 📱 快速开始

### 方法一：使用启动脚本（推荐）
1. 双击运行 `D:\666\start-universal.bat`
2. 等待服务器启动完成
3. 打开浏览器访问：`http://localhost:8888`

### 方法二：手动启动
1. 打开命令提示符（CMD）
2. 输入：`cd D:\666`
3. 输入：`node video-server-universal.js`
4. 打开浏览器访问：`http://localhost:8888`

## 🌐 访问地址

- **手机访问**：http://localhost:8888
- **电脑访问**：http://localhost:8888
- **自动适配**：系统会自动检测您的设备类型，显示最适合的界面

## 🚀 系统要求

- **操作系统**：Windows 10/11
- **Node.js**：已安装（版本 14+）
- **浏览器**：Chrome、Firefox、Safari、Edge等现代浏览器

## ⚠️ 注意事项

1. 这是一个演示版本，所有数据都是模拟的
2. 视频上传功能仅作演示，不会真正保存文件
3. 建议在本地网络中使用，以获得最佳体验
4. 服务器默认运行在8888端口

## 🛠️ 故障排除

### 如果页面无法访问：
1. 检查Node.js是否已安装
2. 确保端口8888未被占用
3. 检查防火墙设置
4. 重新启动服务器

### 如果页面显示异常：
1. 刷新浏览器页面
2. 清除浏览器缓存
3. 尝试使用其他浏览器

## 📞 技术支持

如有问题，请检查：
1. Node.js版本是否兼容
2. 网络连接是否正常
3. 浏览器是否支持现代JavaScript
