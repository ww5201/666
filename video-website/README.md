# 视频网站项目

一个类似腾讯视频、哔哩哔哩的视频网站平台，支持视频上传、播放、搜索、评论等功能。

## 项目特性

- 🎬 视频上传和播放
- 👤 用户注册和登录
- 🔍 视频分类和搜索
- 💬 评论和互动
- 📱 个人主页和收藏
- 🎨 响应式设计

## 技术栈

### 前端
- React 18
- Ant Design
- Video.js
- React Router

### 后端
- Node.js
- Express
- MongoDB
- Multer (文件上传)
- FFmpeg (视频处理)

### 部署
- Vercel (前端)
- Railway (后端)
- AWS S3 (视频存储)

## 项目结构

```
video-website/
├── frontend/          # 前端React应用
├── backend/           # 后端Node.js应用
├── database/          # 数据库配置和脚本
├── uploads/           # 视频上传临时目录
└── README.md         # 项目说明
```
