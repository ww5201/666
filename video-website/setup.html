🎬 视频网站配置

只需 2 分钟，配置云端存储

1

注册 Cloudinary

打开 cloudinary.com，点击「Sign up for free」
用邮箱注册（或 GitHub 登录），注册完成后进入 Dashboard

2

获取 Cloud name

在 Dashboard 首页能看到 Cloud name，复制它

3

创建 Upload Preset

点击左侧「Settings」→「Upload」→ 往下找到「Upload presets」→「Add upload preset」
Preset name 填：video_unsigned
Signing mode 选：Unsigned
点击「Save」

💡 如果提示需要升级，选择免费计划即可。Unsigned preset 不需要付费。
跳过配置，直接使用本地模式 →
