🚀 快速开始部署

推荐使用GitHub Pages + Netlify方案，简单易用且功能强大！

1. 创建GitHub仓库

git init && git add . && git commit -m "Initial commit"

2. 连接到Netlify

访问 https://netlify.com → "New site from Git" → 连接GitHub仓库

3. 获得免费域名

Netlify会自动分配一个随机域名，也可以绑定自定义域名
