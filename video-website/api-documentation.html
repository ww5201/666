🎥 视频相关接口

GET获取视频列表

GET /api/videos
获取视频列表，支持分页、分类筛选和搜索功能
参数说明
参数名 类型 必需 说明
page number 否 页码，默认1
limit number 否 每页数量，默认10
category string 否 分类筛选
search string 否 搜索关键词
响应示例
{
  "videos": [
    {
      "_id": "1",
      "title": "示例视频 1",
      "description": "这是一个示例视频",
      "url": "https://sample-videos.com/zip/10/mp4/SampleVideo_1280x720_1mb.mp4",
      "thumbnail": "https://via.placeholder.com/300x200?text=视频封面",
      "duration": 120,
      "category": "科技",
      "tags": ["教程", "技术"],
      "views": 1000,
      "likes": 50,
      "uploader": { "username": "示例用户" },
      "uploadDate": "2026-06-13T12:00:00.000Z"
    }
  ],
  "totalPages": 1,
  "currentPage": 1,
  "total": 1
}

GET获取单个视频

GET /api/videos/:id
获取单个视频详情，增加观看次数
路径参数
参数名 类型 必需 说明
id string 是 视频ID

category string 是 视频分类
tags string 否 标签，逗号分隔

  "thumbnail": "/uploads/1623584400000-123456789.jpg",
  "duration": 180,
  "category": "科技",

  "views": 0,

    "_id": "60d5ec9f9b1d8c001f8e4a7a",
    "username": "用户名"

}

👤 用户相关接口

POST用户注册

POST /api/users/register
用户注册，创建新账户
请求体
参数名 类型 必需 说明
username string 是 用户名
email string 是 邮箱地址
password string 是 密码
请求示例
{
  "username": "testuser",
  "email": "test@example.com",
  "password": "test123"
}
响应示例
{
  "message": "注册成功",
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "id": "60d5ec9f9b1d8c001f8e4a7b",
    "username": "testuser",
    "email": "test@example.com",
    "avatar": "",
    "role": "user"
  }
}

🔐 认证相关接口

认证说明

所有需要认证的接口都需要在请求头中添加：

Authorization: Bearer <token>

其中 <token> 是用户登录后返回的JWT令牌。

认证请求示例
fetch('/api/users/me', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
  }
})
